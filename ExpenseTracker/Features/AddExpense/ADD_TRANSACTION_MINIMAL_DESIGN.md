# AddTransactionView - Минималистичный редизайн

## ✅ Выполненные изменения

Убраны тяжелые тени со всех компонентов (кроме клавиатуры) и применен современный минималистичный дизайн.

---

## 🎨 Ключевые улучшения

### 1. Type Picker - Сегментированный контрол
**До:** Две отдельные кнопки с тенями  
**После:** Классический iOS segmented control

```swift
HStack(spacing: 0) {  // Без gap между кнопками
    ForEach([.expense, .income]) { type in
        // Активная кнопка - цветной фон
        // Неактивная - прозрачная
    }
}
.padding(4)
.background(Color(.secondarySystemGroupedBackground))  // Общий фон
```

**Улучшения:**
- ✅ Убраны тени
- ✅ Стиль как у нативного UISegmentedControl
- ✅ Меньше визуального шума
- ✅ Плавная анимация переключения

---

### 2. Amount View - Акцент на сумме
**До:** Белая карточка с тенью  
**После:** Цветной фон с индикатором типа

```swift
VStack {
    // Индикатор: точка + текст
    HStack {
        Circle().fill(color)  // Зеленая/красная точка
        Text("Доход/Расход")
    }
    
    // Крупная сумма
    HStack {
        Text("+ / -")  // Знак
        Text("0")      // Сумма (52pt!)
        Text("₸")      // Валюта
    }
}
.background(Color.red/green.opacity(0.08))  // Легкий цветной фон
```

**Улучшения:**
- ✅ Увеличена сумма: 48→52pt
- ✅ Цветной фон вместо белого
- ✅ Индикатор типа сверху
- ✅ Убраны тени
- ✅ Символ ₸ смещен вверх для баланса
- ✅ Больше focus на цифрах

---

### 3. Categories - Горизонтальный скролл
**До:** 4 категории + кнопка "Ещё"  
**После:** Все категории в горизонтальном скролле

```swift
VStack {
    Text("Категория")  // Заголовок секции
    
    ScrollView(.horizontal) {
        HStack {
            ForEach(categories) { category in
                // Все категории доступны
            }
        }
    }
}
```

**Улучшения:**
- ✅ Все категории сразу видны
- ✅ Нет необходимости в "Ещё"
- ✅ Горизонтальный скролл
- ✅ Заголовок секции "Категория"
- ✅ Убраны тени
- ✅ Кольцо вокруг выбранной категории
- ✅ Увеличен размер: 48→54pt

**Выбранная категория:**
- Полный цвет
- Белая иконка
- Цветное кольцо вокруг (outline)

**Невыбранная категория:**
- Светлый цветной фон (opacity 0.15)
- Цветная иконка
- Без кольца

---

### 4. Details - Секции с заголовками
**До:** Два поля без заголовков  
**После:** Секции с лейблами

```swift
VStack(spacing: 16) {
    // Дата
    VStack(alignment: .leading) {
        Text("Дата")  // Заголовок
        
        HStack {
            Image("calendar")
            DatePicker(...)
        }
        .background(Color(.systemBackground))
    }
    
    // Комментарий
    VStack(alignment: .leading) {
        Text("Комментарий")  // Заголовок
        
        HStack {
            Image("text.alignleft")
            TextField(...)
        }
        .background(Color(.systemBackground))
    }
}
```

**Улучшения:**
- ✅ Заголовки секций
- ✅ Убраны тени
- ✅ Убраны обводки
- ✅ Чистый белый фон
- ✅ Увеличены paddings
- ✅ Уменьшено lineLimit: 3-5 → 2-4
